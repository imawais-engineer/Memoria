# Memoria

Personal AI with long-term memory — built for the **Qwen Cloud Hackathon**.

Memoria uses Qwen (via DashScope) for memory extraction, embedding, retrieval, forgetting, and consolidation so an agent can personalize responses across sessions.

**[docs/MEMORIA_DEVELOPMENT_ROADMAP.md](docs/MEMORIA_DEVELOPMENT_ROADMAP.md)**
